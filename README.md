# 夕阳伴侣·银龄AI伴伴

**产品需求文档（PRD）v2.0**

面向中国老年人市场的 AI 驱动移动应用 | Android & iOS 双平台

---

## 项目简介

"夕阳伴侣·银龄AI伴伴"定位为中国老年人的"AI生活管家"，通过大型语言模型（LLM）技术驱动，以情感陪伴为核心，覆盖老年人社交、健康、娱乐、出行、购物五大生活场景。
